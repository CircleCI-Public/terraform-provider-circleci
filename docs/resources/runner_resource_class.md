# circleci_runner_resource_class (Resource)

Manages a CircleCI self-hosted runner resource class. Resource classes define a pool of self-hosted runners within an organization namespace.

## Example Usage

```terraform
resource "circleci_runner_resource_class" "example" {
  organization_id = "00000000-0000-0000-0000-000000000000"
  resource_class  = "my-namespace/my-runner"
  description     = "My self-hosted runner"
}
```

## Schema

### Required

- `organization_id` (String) The organization id.
- `resource_class` (String) The resource class name in `namespace/name` format (e.g. `myorg/myrunner`). Changing this value forces a new resource to be created.

### Optional

- `description` (String) Description of the runner resource class.
- `force_delete` (Boolean) If true, deletes the resource class even if it has associated tokens.

### Read-Only

- `id` (String) Unique identifier (UUID) of the runner resource class.

Import is supported using the resource class name in `namespace/name` format:
