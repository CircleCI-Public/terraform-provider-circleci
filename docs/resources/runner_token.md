# circleci_runner_token (Resource)

Manages a token for a CircleCI self-hosted runner resource class. Tokens are used by self-hosted runner agents to authenticate with CircleCI.

> **Note:** The token value is only available at creation time. After the initial `terraform apply`, CircleCI does not expose the token value via the API and it will not appear in subsequent reads. Store the token value securely (e.g., in a secrets manager) immediately after creation.

## Example Usage

```terraform
resource "circleci_runner_token" "example" {
  organization_id = "00000000-0000-0000-0000-000000000000"
  resource_class  = "my-namespace/my-runner"
  nickname        = "my-runner-token"
}

output "runner_token" {
  value     = circleci_runner_token.example.token
  sensitive = true
}
```

## Schema

### Required

- `nickname` (String) A human-readable label for the token. Changing this value forces a new resource to be created.
- `organization_id` (String) The ID of the organization that owns the resource class. Changing this value forces a new resource to be created.
- `resource_class` (String) The resource class this token grants access to, in `namespace/name` format (e.g. `myorg/myrunner`). Changing this value forces a new resource to be created.

### Read-Only

- `created_at` (String) The time at which the token was created.
- `id` (String) Unique identifier (UUID) of the runner token.
- `token` (String, Sensitive) The token value used to authenticate a runner agent. Only available at creation time — this value is not returned by the API on subsequent reads and will be empty after an import.

## Import

Import is supported using `resource_class/token_id` (e.g., `my-namespace/my-runner/550e8400-...`):

```shell
terraform import circleci_runner_token.example "my-namespace/my-runner/<token_id>"
```

> **Warning:** After import, the `token` value will be empty because the CircleCI API does not return token values after creation. Importing a runner token is only useful for tracking the token's lifecycle in Terraform state. The actual token value cannot be recovered.
